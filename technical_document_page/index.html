Introduction Variables Functions Loops Arrays
Introduction

JavaScript is a programming language used to create interactive websites.

It works together with HTML and CSS.

console.log("Hello World");
  • Easy to learn
  • Runs in browsers
Variables

Variables are used to store data.

You can declare variables using let, const, or var.

let name = "John";
  • let
  • const
Functions

Functions perform reusable tasks.

They help organize code better.

function greet() { return "Hello"; }
  • Reusable
  • Organized code
Loops

Loops repeat code multiple times.

The for loop is commonly used.

for(let i = 0; i < 5; i++) { console.log(i); }
  • for loop
  • while loop
Arrays

Arrays store multiple values in one variable.

Each value has an index number.

let fruits = ["Apple", "Banana", "Orange"];
  • Store lists
  • Indexed values
